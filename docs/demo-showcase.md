# 禾语 AI 下午演示指引

## 演示入口

- 首页：`http://127.0.0.1:8000/`
- 农户内容经营：`http://127.0.0.1:8000/create/?lang=zh-CN`
- 团队工作台：`http://127.0.0.1:8000/workspace/`

第一次使用先双击 `安装禾语AI.bat`，以后只需双击 `启动禾语AI.bat`。演示结束后双击 `停止禾语AI.bat`。

## 推荐演示路线

1. 从首页进入“开始一次内容经营”。
2. 点击“当季番茄”，自动填入农户身份、经营目标、产品信息、发布平台和内容风格。
3. 查看平台给出的选题信号与适配判断。
4. 对比“实用吸睛、人物故事、轻松反差”三条创意路线。
5. 打开其中一条短视频方案，展示标题、前三秒钩子、完整口播、手机分镜、封面文案、背景乐方向和拍摄提示。
6. 继续查看直播话术与七天运营计划。
7. 登录后把方案保存到团队工作台，展示方案预览、编辑、复制和版本记录。
8. 切换简体中文、香港繁体中文和英文，展示三语界面。

## 推荐讲解重点

- 农户不需要学习 Prompt，选择身份、目标、产品和平台后即可开始。
- 热点不是生硬塞入脚本，而是先判断与产品、受众、平台、时效和拍摄条件是否适配。
- 同一个农产品会得到三条风格不同、可以由手机完成拍摄的路线。
- 生成结果不只是一段文案，而是从选题、脚本、拍摄、直播到七天运营的连续工作流。
- 无 API Key 时也能稳定演示；后续可以连接国产 OpenAI-compatible 模型。
